# Cyber Security Toolkit

A Python-based Cyber Security Toolkit that provides multiple security and networking utilities in a single project.

## Features

- Password Generator
- Password Strength Checker
- SHA-256 Hash Generator
- TCP Port Scanner
- DNS Lookup Tool
- WHOIS Lookup Tool
- File Integrity Checker

## Technologies Used

- Python 3
- Socket Programming
- Hashlib
- Python-WHOIS

## Project Structure

```text
Cyber-Security-Toolkit/
├── main.py
├── password_generator.py
├── password_strength_checker.py
├── hash_generator.py
├── port_scanner.py
├── dns_lookup.py
├── whois_lookup.py
└── file_integrity_checker.py
```

## Installation

1. Clone the repository.
2. Install the required dependency:

```bash
pip install python-whois
```

## Usage

Run the toolkit using:

```bash
python main.py
```

Select the desired tool from the menu and follow the on-screen instructions.
